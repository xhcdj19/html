Ejercicio 5

3.5 Escriba un elemento (o elementos) de HTML5 para realizar cada una de las siguientes tareas:
    a) Se pidió a los estudiantes que clasifiquen los alimentos en la cafetería, en una escala del 1 al 10. Use un elemento meter con texto a su izquierda y a su derecha para indicar que la calificación promedio fue 7 de 10.
    < p >1 < meter min = "1" max = "10" value = "7" >< /meter> 10 < /p>


    b) Cree un elemento details que muestre el texto "Resultados de la encuesta" en el elemento summary, deberá aparecer un párrafo que explique sobre la encuesta.
     < details > < summary > Resultados de la encuesta < /summary> < p > La calificación que se le dio a la cafeteria en la escala del 1 al 10. Fue 7. < /p > < /details >
    c) Cree un elemento input text para el primer nombre. El elemento deberá recibir el enfoque de manera automática al momento de desplegar el formulario en un navegador.
     < p > < label > Nombre: < input type = "text" autofocus /> < /label > < /p >
    d) Modifique la parte (c) de modo que se elimine el elemento label y use texto en un elemento placeholder en el elemento input.
     < p > < input type = "text" placeholder = "Nombre" autofocus /> < /p >
    e) Use un elemento datalist para proveer una lista autocomplete para cincos estados.
    < input type = "text" id = "estado" autocomplete = "on" list = "listadeEstados" />
         < datalist >
              < option value = "Veracruz" >
              < option value = "Puebla" >
              < option value = "Chihuahua" >
              < option value = "Oaxaca" >
              < option value = "Chiapas" >
         < /details >
    f) Cree un elemento input range que permita al usuario seleccionar un número del 1 al 100.
    < label > Range: 1 < input type = "range" min = "1" max = "20" value = "10" /> 100 < /label >
    g) Especifique que no debe permitirse el atributo autocomplete en un formulario. Muestre sólo la etiqueta de apaertura del formulario.
    < form action = "post" autocomplete = "off" >
    h) Use un elemento mark para resaltar el segundo enunciado del siguiente párrafo:
    < p >Se pidió a los estudiantes que calificaran los alimentos en la cafetería. En una escala del 1 al 10. El resultado promedio fue 7. < /p >
    < p >Se pidió a los estudiantes que calificaran los alimentos en la cafetería.< mark > En una escala del 1 al 10. El resultado promedio fue 7.< /mark > < /p >
